Tank U
A Phaser Tower Defense Game

Lucas McDaniel
July 7, 2015

Project Goals:

  • Independent towers that rotate and shoot enemy tanks when in range
  • Waves of tanks with varying health that follow a predetermined path
  • An exchange system to activate and upgrade each tower
  • Game music, sound effects, and animations

Time for a live demo!

Lessons Learned:

  • Trust the process at NSS
  • Phaser is awesome, but the learning curve is steep
  • Finishing your first capstone and being proud of it feels amazing
  • Finishing version 1.0 is only the beginning
  • If at all possible, don't have a baby right before capstone
